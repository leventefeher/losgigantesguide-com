I run these classes myself, so I'll be straight with you: they're small (max 12 people), informal, and focused on actually being able to dance socially — not on perfecting technique for competitions.

## What's covered

**Beginner block (first 45 min):** Basic salsa on-1 footwork, partner connection, three or four core patterns. If you've never danced before, this is where you start.

**Intermediate block (second 45 min):** Body movement, leading and following, turn patterns, a bachata basic. If you've been coming a few weeks or have danced before, jump straight in here.

## Practical details

- No partner needed — we rotate
- Wear shoes with a smooth sole (not trainers if you can help it)
- Cost: €10 per session or €35 for a 4-class block
- First class free if you want to try before committing

## Important

Schedules change. **Always confirm via WhatsApp before attending** — cancellations happen when I'm travelling or the hall is booked for another event.
